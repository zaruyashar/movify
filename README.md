# 🎥 Movify: A Catalog Management System

Movify is a comprehensive, N-tier movie management application developed with a focus on clean architecture and modern .NET development practices. The project integrates robust administrative controls with a user-friendly public interface.

## Introduction
Movify serves as a centralized platform for managing a movie archive. It provides administrators with a powerful dashboard for CRUD operations, data reporting, and system maintenance, while offering general users a seamless browsing experience to search the database and explore cinema content.

## Tech Stack
*   **Backend:** C# .NET Core, Entity Framework Core (Code-First)
*   **Database:** MSSQL Server
*   **Frontend (Admin):** HTML5, CSS3, Bootstrap, JavaScript, C3.js (for data visualization)
*   **Frontend (Public):** Movify Template, Custom Responsive Design
*   **Features:**
    *   Dynamic Global Search
    *   PDF/Excel Data Export for management tables
    *   Automated To-Do List integration
    *   Responsive, cross-template design
